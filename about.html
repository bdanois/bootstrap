So far I've learned different aspects of HTML. I now have a broader understanding of which elements manipilate how a website looks. Some elements I am more familiar with include the head and body tags, which are the main elements of any HTML website, along with tags such as title, p, h1. there are also various other useful tags including img, hr, ul, ol, li and br. The entire purpose of HTML is to add content. Elements of design belong to CSS

CSS

CSS or, Cascading Style Sheets is the styling element responsible for the visual appeal of the content created in HTML. CSS is capable of a lot, and libraries such as Bootstrap and JQuery are only a few components capable of manipulating the style elements for an even more appealing website
